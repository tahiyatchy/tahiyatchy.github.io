This Initiative - Project Overview

Overview of the Interactive Ethical Human Subjects Research Resource

Application Description and Purpose

The Interactive Ethical Human Subject Research Resource is an educational platform designed to help students, researchers, and academic professionals understand and apply ethical principles in human subjects research. The platform provides comprehensive learning materials, interactive assessments, and realistic ethical dilemma scenarios based on established frameworks including The Belmont Report, HIPAA regulations, and IRB guidelines.

Primary Purpose

The primary purpose is to create an engaging, accessible resource that prompts prospective researchers to think in terms of real-world scenarios over answers in a vacuum. It is also meant to direct researchers to ask the right questions about their own research to ensure all boxes are checked in the ethics department.

Intended Users

  • University Students: This resource is meant to be a dynamic resource for novice researchers in human subject research.
  • Professors and IRB: A place to direct students and prospective researchers to ensure they have the chance to assess their research themselves before seeking higher guidance.

Website Content Overview

  • Learning Modules: Guides on Belmont Report principles, conflicts of interest, IRB regulations, and data privacy
  • Knowledge Assessments: Interactive quizzes to test understanding of ethical guidelines
  • Ethical Scenarios: Multi-step realistic scenarios that challenge users to apply ethical principles
  • Resource Library: Collection of official documents, guidelines, and reference materials
  • Interactive Tools: Decision-making aids and ethical assessment checklists

Dynamic Functionality


  • Home Page: Lazy loading sections with fade-in animations, AJAX-powered clinical trials data loading from external API, real-time content visibility tracking
  • This Initiative (Project Overview): Scroll-triggered animations, interactive timeline with progress visualization, local storage for user viewing history
  • Learn Page: Interactive educational modules with expandable/collapsible sections, progress tracking with visual indicators, bookmarking functionality using local storage
  • Test Page: Dynamic quiz generation with randomized questions, real-time scoring and progress visualization, form validation for quiz submissions, results persistence in local storage
  • Interactive Page: Scenario-based learning with branching decision paths, real-time ethical dilemma simulations, state management for user choices and consequences, progress tracking through multi-step scenarios


Page Specifications


Home Page (index.html)

  • Audience: Students, researchers, and educators in human subjects research
  • Content: Welcome message, real clinical trials data, about section, disclaimers, terms, resources
  • User Data Entry: No data entry forms
  • Interactive Elements: Navigation links, lazy-loaded sections, dynamic clinical trials data loading

This Initiative (project_overview.html)
  • Purpose: Explain the project goals, methodology, and development process
  • Audience: Educators, project stakeholders, and curious users
  • Content: Project background, design philosophy, development timeline, team information
  • User Data Entry: No data entry forms
  • Interactive Elements: Interactive timeline, expandable sections, progress indicators
  • Actions: Scroll-triggered animations, timeline navigation, section expansion/collapse

Learn Page (learn.html)
  • Purpose: Educational content delivery about ethical research principles
  • Audience: Students and researchers seeking to learn ethical guidelines
  • Content: Belmont Report principles, HIPAA guidelines, informed consent, vulnerable populations
  • User Data Entry: Interactive knowledge checks and self-assessment questions
  • Validations: Answer validation for knowledge check questions
  • Interactive Elements: Expandable content sections, progress trackers, interactive examples, navigation buttons
  • Actions: Content section expansion, progress tracking, bookmarking, knowledge check validation

Test Page (test.html)

  • Content: Branching scenarios, ethical dilemmas, consequence explanations
